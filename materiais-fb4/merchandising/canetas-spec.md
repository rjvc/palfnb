# Canetas Personalizadas — Palestra F&B 4.0

## Especificações Técnicas

| Campo | Valor |
|-------|-------|
| Modelo | Caneta esferográfica metálica com clip |
| Cor do corpo | Azul escuro (#1B365D) ou preto mate |
| Detalhes | Clip e anel em dourado (#C8963E) |
| Tinta | Azul |
| Mecanismo | Rotativo (twist) ou click |
| Personalização | Gravação a laser no corpo da caneta, cor dourada |

## Arte da Personalização

**Gravação a laser no corpo (1 linha ou 2 linhas):**

Opção A (1 linha):
```
GRUPO CASTRO CONSULTORIAS
```

Opção B (2 linhas):
```
GRUPO CASTRO
F&B 4.0
```

## Quantidade
- 100 unidades (80 participantes + margem extra)
