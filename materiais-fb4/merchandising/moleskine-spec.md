# Moleskine Personalizado — Palestra F&B 4.0

## Especificações Técnicas

| Campo | Valor |
|-------|-------|
| Formato | A5 (148 × 210 mm) |
| Capa | Capa dura, forrada a tecido/couro sintético |
| Cor da capa | Azul escuro (ref. #1B365D) |
| Páginas | 96 folhas (192 páginas), papel creme 90g/m² |
| Fechamento | Elástico preto |
| Marcador | Fita de cetim dourada (#C8963E) |
| Personalização | Gravação a quente (hot stamping) dourada na capa frontal |

## Arte da Capa Frontal

**Gravação dourada (hot stamping) na capa:**

```
┌──────────────────────────────┐
│                              │
│     GRUPO CASTRO             │
│     CONSULTORIAS             │
│     ─────────────            │
│     F&B 4.0                 │
│                              │
└──────────────────────────────┘
```

- Logo "GRUPO CASTRO CONSULTORIAS" em Montserrat SemiBold, centralizado, 5 mm do topo
- Linha decorativa horizontal dourada
- "F&B 4.0" em Montserrat Bold, centralizado abaixo da linha
- Hashtag "#F&B4ponto0" opcional na base

## Quantidade
- 80 unidades (para oferta aos participantes da palestra)
