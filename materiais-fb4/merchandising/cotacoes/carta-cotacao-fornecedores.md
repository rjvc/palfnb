## 2. Itens a Cotar

### 2.1 Moleskine Personalizado — 80 unidades

| Característica | Especificação |
|----------------|---------------|
| Formato | A5 (148 × 210 mm) |
| Capa | Capa dura, forrada a tecido/couro sintético |
| Cor da capa | Azul escuro (#1B365D) |
| Páginas | 96 folhas (192 páginas), papel creme 90g/m² |
| Fechamento | Elástico preto |
| Marcador | Fita de cetim dourada (#C8963E) |
| Personalização | Hot stamping (gravação a quente) dourada na capa frontal |
| Arte | "GRUPO CASTRO CONSULTORIAS" + "F&B 4.0" (conforme ficheiro de arte) |

### 2.2 Caneta Esferográfica Personalizada — 100 unidades

| Característica | Especificação |
|----------------|---------------|
| Modelo | Caneta esferográfica metálica com clip |
| Cor do corpo | Azul escuro (#1B365D) ou preto mate |
| Detalhes | Clip e anel em dourado (#C8963E) |
| Tinta | Azul |
| Mecanismo | Rotativo (twist) ou click |
| Personalização | Gravação a laser dourada no corpo |
| Arte | "GRUPO CASTRO CONSULTORIAS" ou "GRUPO CASTRO / F&B 4.0" |
